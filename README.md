# 🌿 Smart Garden Dashboard

Dashboard pemantauan kebun cerdas berbasis IoT untuk 
modernisasi pertanian rooftop dan ekonomi sirkular kampung.

## Fitur
- Pemantauan sensor real-time (suhu, kelembapan, pH, nutrisi)
- Kontrol pompa air jarak jauh
- Status otomatis berdasarkan ambang batas sensor

## Tech Stack
- React + Vite
- Tailwind CSS
